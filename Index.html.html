CSS Base

About CSS Base

It's a common practice to apply CSS to a page that styles elements such that they are consistent across all browsers. We offer two of the most popular choices: normalize.css and a reset. Or, choose Neither and nothing will be applied.

Vendor Prefixing

About Vendor Prefixing

To get the best cross-browser support, it is a common practice to apply vendor prefixes to CSS properties and values that require them to work. For instance -webkit- or -moz-.

We offer two popular choices: Autoprefixer (which processes your CSS server-side) and -prefix-free (which applies prefixes via a script, client-side).

        return isNaN(n) ? 0 : n;

        return;

        div.className = 'cart-item';

      <div class="cart-item-info">
        <div class="cart-item-cat">${item.cat}</div>
        <div class="cart-item-name">${item.nome}</div>
        ${item.preco ? `<div style="font-size:11px;color:var(--pink-deep);margin-top:3px;">${item.preco}</div>` : ''}
        <div class="cart-item-qty" style="margin-top:8px;">
          <button class="qty-btn" data-idx="${idx}" data-op="minus">−</button>
          <span class="qty-num">${item.qty}</span>
          <button class="qty-btn" data-idx="${idx}" data-op="plus">+</button>
        </div>
      </div>
      <button class="cart-item-remove" data-idx="${idx}" aria-label="Remover">✕</button>

          else {
            carrinho[i].qty--;
            if (carrinho[i].qty <= 0) carrinho.splice(i, 1);
          }
          renderCarrinho();
        });

        btn.addEventListener('click', () => {
          carrinho.splice(parseInt(btn.dataset.idx), 1);
          renderCarrinho();
        });
      });
